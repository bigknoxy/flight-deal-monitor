Key Features

🔍 Multi-Source Search

Aggregates flight data from fli (free), SearchAPI, and Duffel with automatic failover.

📉 Smart Deal Detection

Identifies flash sales (≥50% off) and mistake fares (≥70% off) using price history.

🤖 Telegram Alerts

Sends formatted flight deal notifications with booking links directly to Telegram.

💾 Smart Caching

6-hour TTL cache reduces API costs by skipping stable price searches.

🔄 Scheduled Scanning

Regular sweeps every 30 min, mistake fare sweeps every 15 min.

🐳 Docker Support

Ready-to-deploy Docker Compose setup with health checks and auto-restart.

Tech Stack

  • Runtime: Python 3.11+ with async support
  • Web Framework: FastAPI (Starlette/Uvicorn)
  • Database: SQLite (default) / PostgreSQL via SQLAlchemy
  • Flight APIs: fli (curl_cffi), SearchAPI, Duffel
  • Notifications: Telegram Bot API
  • Infrastructure: Docker, Docker Compose, Kubernetes-ready
