Key Features

🔍 Multi-Source Search

fli (free) → SearchAPI ($4/1K) → Duffel backup with automatic failover.

📉 Smart Deal Detection

Flash sales ≥50% off, mistake fares ≥70% off using price history analysis.

🤖 Telegram Alerts

Instant notifications with direct booking links to Telegram.

💾 Smart Caching

6-hour TTL reduces redundant API calls and costs.

🔄 Scheduled Scanning

30-min regular sweeps, 15-min mistake fare priority scans.

🐳 Docker Ready

Production Docker Compose with health checks and auto-restart.
