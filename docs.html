Quick Install

# Clone and install git clone https://github.com/bigknoxy/flight-deal-monitor.git cd flight-deal-monitor pip install -r requirements.txt
# Configure (required) cp config/.env.example .env # Add SEARCHAPI_API_KEY, TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID
fli note: Free but rate-limited. SearchAPI ($4/1K) recommended for reliability.

Environment Variables

VariableRequired
SEARCHAPI_API_KEYYes (reliable fallback)
TELEGRAM_BOT_TOKENYes
TELEGRAM_CHAT_IDYes
DUFFEL_API_TOKENOptional backup

API Usage

from fli.search import SearchFlights searcher = SearchFlights() filters = FlightSearchFilters( passenger_info=PassengerInfo(adults=1), flight_segments=[...], ) results = searcher.search(filters, top_n=10)

python -m app.main # Server runs at http://localhost:8000
